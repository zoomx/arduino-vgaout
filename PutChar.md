## put\_char(x,y,c) ##
Print a character at given coordinate.

## Use ##
put\_char(x,y,c)

  * Arguments
    * unsigned char x : x-coordinate 0 - 7
    * unsigned char y : y-coordinate 0 - 13
    * unsigned char c : ASCII code character and original ( file : CG\_xx.h )
